Summary: Example non-article blog page

Templated pages that are related to the blog, but are not indexed articles, can be present in any blog sub-directory except articles; tags and dates should probably be avoided.

If, like this page, they use the blog navigation, then they should be in a subdirectory of blog. This page is located in blog/config/.
