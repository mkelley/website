Blog tags

Blog tags can (probably) use any characters allowed as a file name, plus spaces and forward slashes. The tag is transformed into a file name with the filequote filter. This filter replaces spaces and slashes with the underscore character. Thus, the index for the tag labeled open spaces will be tags/open_spaces.html.
